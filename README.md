# Sistema de Gestión de Gimnasio

## Descripción

Prototipo web de alta fidelidad para la administración de un gimnasio.

## Funcionalidades

* Gestión de clientes
* Gestión de entrenadores
* Gestión de membresías
* Gestión de rutinas
* Control de asistencia
* Dashboard con estadísticas

## Tecnologías utilizadas

* HTML
* CSS
* JavaScript
